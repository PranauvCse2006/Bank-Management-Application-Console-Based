# Bank Management Application in C++

## Description

A console-based Bank Management System developed using C++ with Object-Oriented Programming and file handling for persistent data storage.

## Features

* Create bank accounts
* Secure login system using PIN
* Deposit money
* Withdraw money
* Check account balance
* View account details
* Transaction history
* Admin panel
* Delete customer accounts
* Persistent storage using text files
* Menu-driven interface

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling
* Text File Storage

## How to Compile

g++ main.cpp Account.cpp -o bank

## How to Run

./bank

## Files

* main.cpp → Main source code
* Account.cpp → Account-related functions
* Account.h → Class declarations
* accounts.txt → Customer account database
* transactions.txt → Transaction records
